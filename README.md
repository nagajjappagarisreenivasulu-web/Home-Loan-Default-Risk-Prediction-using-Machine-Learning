# Home Loan Default Prediction Project

## Project Description
Predict home loan default probability using ML models (Logistic Regression, Random Forest, XGBoost). Includes feature engineering, risk scoring, and model comparison visualization.

## Technologies
- Python
- Pandas, NumPy
- Scikit-learn, XGBoost
- Matplotlib, Seaborn

## Project Structure
- `notebooks/` : Jupyter notebook with code
- `data/` : input CSV files
- `models/` : saved ML models
- `README.md` : project description

## How to Run
1. Install dependencies from `requirements.txt`
2. Open notebook in Jupyter
3. Run all cells to reproduce results
